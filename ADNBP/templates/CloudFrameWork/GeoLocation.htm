GeoLocation API

You can allow users can share their locations to interact with your services.

For JavaScript API include:

  • IN THE HTML HEAD Section: <script src="/ADNBP/js/adnbpcloudframework.js"></script> and you will can to use two simple Javascript Functions.
    - setGeoLocationByBrowser(); getGeoLocationByBrowser();

    • If you want to use Google Maps API.:

  • IN config/config.php SET:
    $this->setConf("GoogleMapsAPI",{GoogleMapsAPI}); // Need a true value
    // Put a valid value from https://console.developers.google.com/project/{project}/apiui/api
    $this->setConf("GooglePublicAPICredential","{GooglePublicAPICredential}"); 

    •         
  •  IN HE HTML HEAD and BODY Section:
    • 
        <?php if(strlen($this->getConf("GooglePublicAPICredential")) && $this->getConf("GoogleMapsAPI")) {?>
            <meta name="viewport" content="initial-scale=1.0, user-scalable=no" />
            <style type="text/css">
              #map-canvas { height: 100% }
            </style>
            <script type="text/javascript"src="https://maps.googleapis.com/maps/api/js?key=getConf("GooglePublicAPICredential")?>&sensor=true">
         <?php } ?>
         ....
         </head>
         ....

         <div id="map-canvas"/>
