Managing DataStore

In GAE, DataStore is a powerfull way to create no SQL data

You have to: from https://console.developers.google.com/project/apps~{dataset-id}/apiui/credential

- Store the private key in '/config/key.p12'

- setting up the conf Vars:

  • $this->setConf('GoogleCloudProjectName','A name for your app engine app');
  • $this->setConf('ServiceAccountEmailAddress','xxx@developer.gserviceaccount.com');
  • $this->setConf('GoogleCloudProjectId','your-app-id');
