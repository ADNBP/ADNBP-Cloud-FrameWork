CloudSQL/Mysql API

Let's see how we can interact with a CloudSQL or Mysql Database.

For CloudSQL connection you have to use a socket:

$this->setConf("dbSocket","/cloudsql/{application}:{dbName}");

For Remote CloudSQL or Mysql have to use a IP:

$this->setConf("dbServer",'{IP}');

Common configuracion:

$this->setConf("dbUser",'{dbUser}'); // In Geanet is adnbp
$this->setConf("dbPassword",'{dbPassw}');
$this->setConf("dbPort",'{dbPort}'); // Only used with dbServer
$this->setConf("dbName",'{dbName}');
